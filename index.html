MESA Installation Guide

Follow these steps to install MESA easily:

  1. Download Git Bash: https://git-scm.com/downloads
  2. Download X Server:
    • For Windows: Download Xming
    • For macOS: Download XQuartz
  3. Download Docker Desktop: https://www.docker.com/products/docker-desktop/
  4. Follow the remaining steps from the MESA-Docker GitHub repository: https://github.com/evbauer/MESA-Docker

Additional Notes

  • Comment out the last line of the ./xxx_dockerMESA.sh file(s) to keep the Docker container running:
    # docker kill mesa_dock
  • Instead of running the non-versioned command, run this versioned command:
    ./win_dockerMESA.sh -v 15140 (Windows)
    ./mac_dockerMESA.sh -v 15140 (macOS)
